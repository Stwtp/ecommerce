How to run this project

website url: http://localhost:3000
api url: http://localhost:5001
db connection Hostname: localhost port: 5432 DBname: playground

1. cd into project
2. run docker-compose up --build -d or docker compose up --build -d
3. cd into webapi-boilerplate/
4. run dotnet ef database update
5. go to web http://localhost:3000
6. login with email: admin@example.com password: Admin123!
