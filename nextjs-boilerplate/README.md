# Next.js Boilerplate

This repository contains a minimal Next.js 13 app router starter with TypeScript, ESLint, and Prettier configured.

## Getting Started

```bash
npm install
npm run dev
```

## Features
- TypeScript support
- App router (pages in `app/`)
- ESLint + Prettier
- Basic `.env.example` for environment variables
